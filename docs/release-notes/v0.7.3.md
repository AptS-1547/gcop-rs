# Release v0.7.3

## Dependencies - Upgrade Core Libraries

This release upgrades core dependencies and adds Claude Code Hub provider example.

### Updated

**Dependency Upgrades**

- `reqwest`: 0.12 → 0.13
- `mockall`: 0.13 → 0.14
- `git2`: → 0.20.4 (security fix for GHSA-j39j-6gw9-jw6h)

**MSRV Update**

- Minimum Supported Rust Version: 1.92.0 → 1.93.0

### Added

**Claude Code Hub Provider Example**

Added Claude Code Hub configuration example in `examples/config.toml.example`, demonstrating how to configure Claude-compatible custom providers.

### Upgrade

```bash
# Homebrew
brew upgrade gcop-rs

# Cargo
cargo install gcop-rs

# pip
pip install --upgrade gcop-rs
```
