# Release v0.7.3

## 依赖更新 - 升级核心库

本次发布升级核心依赖，并新增 Claude Code Hub provider 配置示例。

### 更新

**依赖升级**

- `reqwest`: 0.12 → 0.13
- `mockall`: 0.13 → 0.14
- `git2`: → 0.20.4 (安全修复 GHSA-j39j-6gw9-jw6h)

**MSRV 更新**

- 最低支持 Rust 版本: 1.92.0 → 1.93.0

### 新增

**Claude Code Hub Provider 示例**

在 `examples/config.toml.example` 中新增 Claude Code Hub 配置示例，演示如何配置 Claude 兼容的自定义 provider。

### 升级

```bash
# Homebrew
brew upgrade gcop-rs

# Cargo
cargo install gcop-rs

# pip
pip install --upgrade gcop-rs
```
