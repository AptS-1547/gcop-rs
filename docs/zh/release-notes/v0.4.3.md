# Release v0.4.3

## 构建系统

### 原生 Python Wheels

PyPI 安装现在使用 **maturin** 构建原生 wheels，取代了之前的 Python 包装器方式。

**之前 (v0.4.2)：** Python 包在首次运行时下载预编译二进制。

**现在 (v0.4.3)：** 为 6 个平台预构建原生 wheels：
- `x86_64-unknown-linux-gnu` (Linux AMD64)
- `aarch64-unknown-linux-gnu` (Linux ARM64)
- `x86_64-apple-darwin` (macOS Intel)
- `aarch64-apple-darwin` (macOS Apple Silicon)
- `x86_64-pc-windows-msvc` (Windows AMD64)
- `aarch64-pc-windows-msvc` (Windows ARM64)

这意味着：
- 更快的安装速度（无需运行时下载）
- 更好的可靠性（安装后无网络依赖）
- 更好的 pip/pipx 集成

```bash
# 使用 pipx 安装（推荐）
pipx install gcop-rs

# 使用 pip 安装
pip install gcop-rs
```

## 文档

### VitePress 迁移

文档已迁移到 **VitePress**，带来以下改进：

- 现代化、快速的文档站点
- 全文搜索支持
- 更好的侧边栏导航
- 改进的移动端体验

### 多语言支持

文档现在同时支持**英文**和**中文**：
- 英文：`/guide/`
- 中文：`/zh/guide/`

### 自动语言重定向

文档首页现在会根据浏览器语言偏好**自动重定向**到对应语言版本。

## 升级说明

从 v0.4.2 升级**无需任何操作**，完全向后兼容。

如果你通过 pip/pipx 安装，重新安装将获得原生 wheel：

```bash
pipx reinstall gcop-rs
```

## 安装

```bash
# Homebrew (macOS/Linux)
brew tap AptS-1547/gcop-rs
brew install gcop-rs

# pipx (Python 用户，推荐)
pipx install gcop-rs

# cargo-binstall (无需编译)
cargo binstall gcop-rs

# cargo install (从源码编译)
cargo install gcop-rs
```

或从 [Releases](https://github.com/AptS-1547/gcop-rs/releases) 下载预编译二进制。
