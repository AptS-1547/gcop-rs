# 关于 gcop-rs

## 项目历史

gcop-rs 是对原 [gcop](https://github.com/Undertone0809/gcop) 项目的 Rust 重写，原项目由 [Undertone0809](https://github.com/Undertone0809) 使用 Python 创建。

原版 gcop 开创了使用 AI 生成有意义的 Git commit message 的理念，让开发者更容易维护一致且描述清晰的提交历史。它证明了 AI 辅助 Git 工作流的价值，并启发了许多开发者。

Rust 版本基于这一基础，旨在提供：
- 通过 Rust 的效率提升性能
- 更低的资源占用
- 更好的可靠性和可维护性
- 扩展功能的同时保留核心理念
